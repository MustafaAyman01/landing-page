# Unifra — صفحة عرض الأنظمة

صفحة هبوط (Landing Page) لعرض أنظمة إدارة المصانع اللي بتتعمل، مع تواصل مباشر عبر واتساب.

## صور الـ Hero (أعلى الصفحة)

كولاج لقطات حقيقية من النظام (لوحة التحكم + الفواتير + كشف الحساب + المنتجات + التقارير) — دي موجودة بالفعل ومش محتاجة أي حاجة، بس لو عايز تغيّرها بعدين استبدل نفس أسماء الملفات دي في `assets/images`:
`hero-dashboard.png` (الأساسية) · `hero-invoices.png` · `hero-statement.png` · `hero-products.png` · `hero-reports.png`

## إزاي تضيف صور وفيديوهات لباقي الصفحة

مفيش أي تعديل في الكود مطلوب. بس ارفع الملفات بنفس الأسماء دي في المكان الصح:

| المكان المطلوب رفعه فيه | اسم الملف المطلوب بالظبط |
|---|---|
| `assets/images/` | `feature-invoices.png` |
| `assets/images/` | `feature-statement.png` |
| `assets/images/` | `feature-stocktake.png` |
| `assets/images/` | `feature-reports.png` |
| `assets/videos/` | `demo-video.mp4` |

- الصور الأفضل تكون سكرين شوت لشاشة النظام (أي مقاس، الصفحة بتظبطها لوحدها).
- الفيديو المفروض يكون بصيغة mp4.
- طول ما الملف مش موجود، الصفحة بتعرض شكل بديل بسيط (Placeholder) بدل ما تظهر صورة مكسورة — يعني تقدر ترفع الصور بالتدريج براحتك من غير ما حاجة "تتكسر" في الصفحة.

## إزاي ترفع صورة/فيديو من على GitHub مباشرة (من غير جهاز كمبيوتر)

1. افتح الـ repo على GitHub
2. ادخل مجلد `assets/images` (أو `assets/videos`)
3. دوس **Add file → Upload files**
4. اسحب الصورة/الفيديو وسمّيه بالظبط بالاسم المطلوب من الجدول فوق
5. دوس **Commit changes**
6. حدّث الصفحة (Refresh) بعد دقيقة أو اتنين وهتلاقي التحديث ظهر

في `index.html` فيه تعليق بعنوان **"قالب تفاصيل نظام"** — انسخ البلوك ده كامل وغيّر فيه العنوان والأيقونة والمشاكل والمميزات ورابط الواتساب، وضيفه بعد النظام الحالي.
